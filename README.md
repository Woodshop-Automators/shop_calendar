# Woodshop Calendar

A low-cost event calendar website powered by Google Sheets and GitHub Pages.

## Quick Start

1. **Set up Google Sheet**: Create a Google Sheet with your events data
2. **Deploy Apps Script**: Use the code in `apps-script.gs` to create a JSON API
3. **Update config.js**: Replace `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE` with your API URL
4. **Deploy**: Push to GitHub and enable GitHub Pages

See [SPEC.md](SPEC.md) for full technical specifications.
